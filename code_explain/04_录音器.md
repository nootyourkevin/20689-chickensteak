## 1. 录音器配置 (`recorder.py`)

```python
class AudioRecorder(QObject):
    """从麦克风采集音频"""

    CHUNK = 1024           # 每次读取的帧数（缓冲区大小）
    CHANNELS = 2           # 立体声（ELF2 设备只支持立体声）
    RATE = 16000           # 采样率 16kHz

    # 信号定义
    recording_started = pyqtSignal()          # 录音开始信号
    recording_finished = pyqtSignal(bytes)    # 录音完成信号（PCM 数据）
    recording_error = pyqtSignal(str)         # 录音错误信号
    audio_level = pyqtSignal(float)           # 音量电平信号（0.0-100.0）
```

**作用**：定义录音参数和信号。

---

## 2. 打开录音设备

```python
def _record_thread(self):
    """后台线程：打开 PyAudio 并循环读取音频帧"""
    import pyaudio

    p = pyaudio.PyAudio()
    stream = p.open(
        format=pyaudio.paInt16,      # 16-bit 整数格式
        channels=self.CHANNELS,      # 立体声
        rate=self.RATE,              # 16kHz 采样率
        input_device_index=1,        # ELF2 使用 card 1 (rockchipnau8822)
        frames_per_buffer=self.CHUNK,
    )
```

**作用**：打开指定的录音设备，`input_device_index=1` 指定 ELF2 的声卡。

---

## 3. 循环录音

```python
    while self._recording:
        try:
            data = stream.read(self.CHUNK, exception_on_overflow=False)
        except Exception:
            break
        self._frames.append(data)  # 把音频数据存入列表

        # 计算 RMS 电平（用于 UI 显示音量条）
        sample_count = len(data) // 2
        samples = struct.unpack(f"<{sample_count}h", data)
        rms = math.sqrt(sum(s * s for s in samples) / sample_count)
        level = min(rms / 32768.0 * 100.0, 100.0)
        self.audio_level.emit(level)  # 发送音量信号给 UI
```

**作用**：持续录音，直到 `_recording` 变为 False。

---

## 4. 录音结束，返回数据

```python
    # 线程退出时返回录音数据
    all_data = b"".join(self._frames)  # 把所有音频帧拼接起来
    self._frames = []
    self._recording = False
    self.recording_finished.emit(all_data)  # 发送信号，把 PCM 数据传出去
```

**作用**：录音结束后，把所有音频帧合并成一个完整的 PCM 数据，通过信号发送出去。

---

## 5. 停止录音

```python
def stop(self):
    """设置停止标志。录音线程会在下一次 read() 后退出。"""
    self._recording = False
```

**作用**：设置标志位，让录音线程在下一次循环时退出。

## 📊 录音参数

| CHUNK | 1024 | 缓冲区大小 |

| 设备索引 | 1 | ELF2 card 1 |

---

## 🔄 录音流程

```
用户按下按键
    ↓
start() → 创建线程，开始录音
    ↓
_record_thread() → 打开设备，循环读取音频
    ↓
用户松开按键
    ↓
stop() → 设置 _recording = False
    ↓
线程退出 → 合并所有音频帧
    ↓
recording_finished.emit(all_data) → 发送信号给 chat_page.py
```

---

## 📋 输出格式

- **数据类型**：`bytes`（原始 PCM）
- **格式**：16-bit signed int, 立体声, 16kHz
- **大小**：每秒 64000 bytes（16000 采样 × 2 通道 × 2 字节）

**这就是为什么 ASR 需要把立体声转单声道的原因！**
