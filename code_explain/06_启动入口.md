## 1. 原版 (`main.py`)

### 1.1 创建 LLM

```python
def create_llm(backend: str):
    """根据命令行参数创建对应的 LLM 适配器"""
    if backend == "cloud":
        api_key = os.environ.get("CLOUD_API_KEY", CLOUD_API_KEY)
        api_url = os.environ.get("CLOUD_API_URL", CLOUD_API_URL)
        return CloudLLM(api_url=api_url, api_key=api_key)
    return MockLLM()
```


## 3. DeepSeek V4 Flash 版本 (`main_deepseek_v4_flash.py`)

### 3.1 创建 LLM（快速响应）

```python
def create_llm(backend: str, model: str = "deepseek-v4-flash"):
    """根据命令行参数创建对应的 LLM 适配器"""
    if backend == "cloud":
        api_key = os.environ.get("CLOUD_API_KEY", CLOUD_API_KEY)
        api_url = os.environ.get("CLOUD_API_URL", CLOUD_API_URL)
        # 极速模式：最小化输出
        return CloudLLM(
            api_url=api_url,
            api_key=api_key,
            model=model,
            max_tokens=30,      # 最少输出（约 20 个英文词）
            temperature=0.1,    # 最低温度，几乎不思考
            top_p=0.3,          # 核采样，减少词汇选择范围
        )
    return MockLLM()
```

---

### 3.2 启动命令

```bash
CLOUD_API_KEY=sk-xxx DISPLAY=:0 PYTHONPATH=src python3 src/main_deepseek_v4_flash.py --voice --asr sensevoice --llm cloud --tts piper
```

```
    ↓

    ↓

```
